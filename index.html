Introduction to HTML

HTML (HyperText Markup Language) is the fundamental building block of the web. It is a markup language—not a programming language—used to define the structure and content of a webpage, such as headings, paragraphs, images, and links. It relies on a system of opening and closing tags, such as and to wrap text and signal how that specific piece of content should behave. While HTML handles the structural foundation of frontend development, it is intentionally basic on its own, meaning you will eventually pair it with CSS to add visual styles and JavaScript to introduce interactive features.
